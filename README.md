# Business Performance & Churn Intelligence Platform

## Overview
This project demonstrates a senior-level Business Intelligence solution that integrates
payments, engagement, and training data to identify churn risks, measure training ROI,
and provide executive-ready insights.

## Business Problem
Leadership lacked a single source of truth to:
- Track active vs inactive paid businesses
- Understand why engagement drops month-over-month
- Measure impact of trainings on business activity
- Proactively identify churn risks

## Solution
Built an end-to-end BI pipeline using SQL and analytics modeling to:
- Create a 360Â° business performance view
- Classify businesses into activity buckets
- Track engagement decline trends
- Measure training effectiveness
- Enable executive dashboards

## Tools Used
- SQL (CTEs, Window Functions, Optimization)
- Excel / Power BI (for visualization)
- GitHub (version control & documentation)

## Key Outcomes
- Identified ~25% businesses at high churn risk
- Reduced manual reporting effort by ~30%
- Provided leadership with actionable insights
